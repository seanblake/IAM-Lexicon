Referring to crypto-systems which use a single key to perform cryptographic operations, or the key itself - i.e. a [[Symmetric key]]

Commonly use of crypto-systems using symmetric keys are generically referred to as:
- [[Symmetric Cryptography]] / [[Symmetric key Cryptography]] for any generic cryptographic operation; or,
- [[Symmetric Encryption]] / [[Symmetric key Encryption]] for encryption (and decryption) operations
