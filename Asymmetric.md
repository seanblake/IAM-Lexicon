Referring to crypto-systems which use a pair of key to perform cryptographic operations, or a key / the pair of keys itself - i.e. an [[Asymmetric key]] / an [[Asymmetric key pair]]

Commonly use of crypto-systems using asymmetric keys are generically referred to as:
- [[Asymmetric Cryptography]] / [[Asymmetric key Cryptography]] for any generic cryptographic operation; or,
- [[Asymmetric Encryption]] / [[Asymmetric key Encryption]] for encryption (and decryption) operations
