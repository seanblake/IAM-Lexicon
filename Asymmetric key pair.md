The two keys within a pair used in [[Asymmetric key Cryptography]], known as either:
- the [[Public key]] which can be shared; or,
- the [[Private key]] which must be kept secret to maintain security

The keys in an asymmetric key pair are intrinsically linked based on the fact that:
- data enciphered using the public key may be deciphered using the private key; and,
- data enciphered using the private key may be deciphered using the public key
