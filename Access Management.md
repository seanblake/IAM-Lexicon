Practically, performing the real-time functions of:
- [[Authentication]]; and,
- [[Authorization]]

Additionally, administration and configuration of the systems which deliver these domains of capability through implementation of technical controls to evaluate access policies and make access decisions, either granting or denying access to resources.

Access Management is related, but distinct from [[Privileged Access Management]], which implements additional capabilities which provide additional security controls to sensitive (i.e. "privileged") activities.
