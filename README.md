# IAM-Lexicon

A dictionary of Identity and Access Management-related terms in [Obsidian](https://www.obsidian.md) markdown.

## Use

### Pre-requisites

- Obsidian installed
- git installed

### Instructions

1. Create a folder to store Obsidian content (i.e.  "Vaults") - e.g. `~\ObsidianVaults\`
2. Open a terminal, cd to the directory you created
3. run `git clone https://github.com/seanblake/IAM-Lexicon`
4. Open Obsidian, select "Open folder as vault" and point it to the folder of the cloned repo - e.g. `~\ObsidianVaults\IAM-Lexicon`
5. Explore and enjoy!
