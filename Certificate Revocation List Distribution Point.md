Abbreviated as:  [[CDP]], or uncommonly [[CRL DP]]

A network location where [[Certification Authorities]] make a [[Certificate Revocation List]] available for querying as part of the verification process of a [[Digital Certificate]].

Hosting a [[CDP]] is one function of a [[Validation Authority]] in a [[Public Key Infrastructure]] (along with hosting [[AIA]], and often an [[OCSP]]).

[[CDP]] usually host the lists on servers using either or both of:
- [[LDAP]]; or,
- [[HTTP]] / [[HTTPS]]
