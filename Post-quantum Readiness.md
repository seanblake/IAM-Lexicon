Abbreviated as [[PQR]]

This describes the act of transitioning from the use of [[Classical cryptography]] to [[Post-quantum Cryptography]].

This is a potentially quite complicated task, where organizations must:
- identify all of their uses of cryptography
- discover when and where cryptography is used
- determine which cryptographic schemes are in use
- determine whether the schemes are fully or partly susceptible to attacks using [[Quantum computing]] and require changes

Once the above is completed, organizations must:
- determine whether products used support [[PQC]]
- verify separate systems which interoperate all support [[PQC]]
- plan the upgrade or replacement of systems which don't support [[PQC]], if applicable
- reconfigure systems to use [[PQC]] 
- coordinate classical-to-PQC migrations for separate systems which need to interoperate
